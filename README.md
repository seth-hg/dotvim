# 使用说明

## 安装

```
git clone https://github.com/seth-hg/dotvim.git .vim
cd ~/.vim
git submodule init
git submodule update
```


## 快捷键

Vim常用快捷键参考[这里](https://www.jianshu.com/p/dde77e3b299f)。

### c-support快捷键



###自定义快捷键

* F2：打开/关闭NERDTree文件窗口

* F3：打开/关闭文件结构视图

* F6：打开/关闭行号显示

* TAB：自动格式化当前行或者选中区域

* Ctrl + P：显示最近打开的文件列表
